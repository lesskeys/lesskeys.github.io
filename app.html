LessKeys-App

The Android application serves as the central control element of the system for apartment owners, property managers and tenants. After installation on the user's mobile phone, he/she must log in with his/her access data.

The most important functionalities of the LessKeys app include:

  • Interaction with NFC chips
  • Key function of the telephone
  • authorization management
  • lock control

The app was developed in Java for Smarthphones with an Android operating system. The communication with the server takes place via REST. An important point is the implementation of NFC. A distinction is made between the HCE mode, for communication between NFC reader and smartphone, and the NFC interface for writing the chips.
