ABOUT


LessKeys is a multi-lock system designed for multi-apartment buildings. The project was started by three students during research for their bachelor thesis at the University of Innsbruck.
The goal of the project is to provide a basic foundation for developing such a system.
Based on the use-case, development was focused on user-privacy, whilest only using state of the art technology, like the "Near Field Communication" standard (NFC).
The project itself is devided into four essential components: android app, server application, lock software and web-app.

OUR THREE COMPONENTS

The system consists of 3 main components, which are connected to each other via the Internet

phone_android

APP

The android-app is the central control element of the system for appartment owners, property managers and tenants.
The main functionalities of the LessKeys-App are opening locks, reading and writing NFC chips, and lock management.

data_usage

SERVER

An application designed as a communication point between the different tiers, while also functioning as the central data storage in combination with a SQL-based database.

SERVER
lock_outline

Lock Software

A program suited to run on a Raspberry Pi with special Modules, like a PN532 NFC reader and a basic LCD display.
In combination with a 5V relais, the SoC can function as a controller for an electric door lock.

WebAPP

We have developed a web application for configuration and basic control of the server software. It also uses the REST interface. An administrator can also view the system log via the Web app. In addition, it has a function that enables ringing at the main entrance door.
