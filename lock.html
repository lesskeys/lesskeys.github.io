About

Lock System

The lock system serves as the central control element for the procedure of unlocking a door. It is best compared to a conventional door lock. Though a major difference are the keys which have authorisations for corresponding doors. As result a key can on one hand be authorised for several doors, but on the other hand, a door can also be unlocked by many different keys. Such a key is represented as a unique amount of data that is transferred to the lock system via NFC either from a smartphone, or a NFC-chip.

Hardware-Components

  • Raspberry Pi single-board computer, responsible for controlling the other hardware-components
  • NFC-Reader, controlling the NFC-communication between lock and smartphone or NFC-Chip
  • LCD display for user feedback
  • 5V relay to control the electronic door lock

Software-components

  • Libnfc
  • libcurl
  • SQlite database

Features

  • Unlocking a door via NFC with the Keyless-App
  • Unlocking a door via NFC with a NFC-chip
  • Remote unlocking a door from a long distance (for example from your workplace)
