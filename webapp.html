LessKeys-Webapp

Info

We have developed a web application for configuration and basic control of the server software. It also uses the REST interface. An administrator can also view the system log via the Web app. In addition, it has a function that enables ringing at the main entrance door.

Technologies

For the web application we use React JS. Since the server application also functions as a backend for this, we do not need a separate backend. Therefore, the web app only has to be hosted on a static web server. We use Yarn as Dependency Manager. Google Firebase is used for communication with the Android app. For this there is a file in which all access data is stored in JSON format. This file is required by the server application to use the Firebase SDK.
