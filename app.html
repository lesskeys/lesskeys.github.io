LessKeys-App

The Android application serves as the central control element of the system for apartment owners, property managers and tenants. After installation on the user's mobile phone, he/she must log in with the given access data.

The app was developed in Java for Smarthphones with an Android operating system. The communication with the server takes place via REST. An important point is the implementation of NFC. A distinction is made between the HCE mode, for communication between NFC reader and smartphone, and the NFC interface for writing the chips.

The most important functionalities of the LessKeys app include:

  • Interaction with NFC chips
  • Key function of the telephone
  • authorization management
  • lock control

The interaction with NFC chips is about being able to describe and read the chips. The writing of the chips is used to create new keys. The purpose of reading the chips is to read in a foreign key found in the residential complex and thus to determine its owner.

The key function of the smartphone is to use the phone as the key for the lock system. The smartphone emulates an NFC chip using HCE.

Authorization management includes viewing, passing on, and editing authorizations for locks in the system. The aim is to make it as easy as possible for users to view authorizations for their own locks and to pass these authorizations on to other users.

Locks should also be able to be accessed via a remote connection. Via button click, the user can easily open the main gate of the apartment building.

Technical part

The Android application is built on Android SDK-Version 26. The minimal SDK-Version is Version 23. We have used several technologies for building our App:

  • Android SDK
  • Gradle Build Tool
  • TapLinx SDK
  • Host Card Emulation
  • Firebase Cloud messaging

Taplinx

Taplinx was developed by Dutch semiconductor manufacturer NXP-Semiconductors. It simplifies working with NFC chips in connection with app development in Android by providing predefined methods. These include writing and reading the NFC chips as well as encryption of the chips. Link to TapLinx: https://www.mifare.net/en/products/tools/taplinx
